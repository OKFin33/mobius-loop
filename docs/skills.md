# Skills

Mobius Loop skills are plain directories with a `SKILL.md` entrypoint.

V0 public skills:

- `mobius-loop`: controller protocol.
- `wiki-retrieval`: read-side recall.
- `wiki-bridge`: knowledge base to project workspace bridge.
- `wiki-fy`: evidence writeback and judgment gates.

Deferred skills:

- `wiki-fy-shift`: scheduled writeback shift.
- workstate/secretary skills.
- node bootstrap and remote mirror skills.

Deferred skills need generic configuration, fake fixtures, and public tests
before they belong in the public core.
